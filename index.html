Program Pendidikan

  • Seragam Siswa Gratis
  • Beasiswa Santri dan Sarjana
  • Insentif Guru Tidak Tetap
  • Internet Gratis Sekolah dan Madrasah
  • Peningkatan Insentif Guru Ngaji

Program Kesehatan

  • Biaya Kesehatan Gratis untuk Imam Masjid dan Guru Ngaji
  • Jampresa Gratis
  • Kesehatan Ibu dan Anak

Program Pertanian

  • Ketersediaan Pupuk Mudah dan Murah

  • Modernisasi Pertanian
  • Kawasan Peternakan Terpadu

Program Infrastruktur

  • Peningkatan dan Pemerataan Infrastruktur Pedesaan
  • Bedah Rumah Tidak Layak Huni

Program Pariwisata

  • Pariwisata Magelang Go Internasional
  • Pengembangan Desa Wisata Berbasis Budaya dan Kretif Lokal

Program UMKM & Ekonomi Kreatif

  • Bantuan Usaha UMKM dan Ekonomi Kreatif
  • Peningkatan Sarana Ekonomi Kreatif Berbasis Digital
  • Fasilitas Pengembangan UMKM dan Ekonomi Kreatif
  • Pembangunan pasar Ekonomi Kreatif
  • Fasilitas Event Nasional dan Internasional

Program Seni & Budaya

  • Pengembangan Wisma Seni dan Budaya

Program Pemuda & Olahraga

  • Beasiswa Pemuda dan Santri Berprestasi
  • Fasilitas Pengembangan Potensi Generasi Z dan Milenial
  • Pengembangan Sport Center dan Event Olahraga
  • Penghargaan Insan Olahraga Berprestasi
  • Pengembangan Eco Sport Tourism

Program Tata Kelola Pemerintahan

  • Meningkatkan Kesejahteraan Perangkat Desa
  • Pemberian Penghasilan Tetap ke-13 dan ke-14 untuk Kepala Desa dan Perangkat Desa
  • Intensif Kelembagaan Desa untuk Ketua RT dan Ketua RW
  • Mengembalikan Kepala Dusun Berbasis Dusun
